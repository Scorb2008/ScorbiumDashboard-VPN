
# Платёжные системы

## ЮКасса

### Настройка

1. Зарегистрируйтесь на [yookassa.ru](https://yookassa.ru)
2. Создайте магазин
3. Получите Shop ID и Secret Key
4. Откройте админ-панель и перейдите в `Telegram -> Платёжные системы`
5. Заполните `Shop ID` и `Secret Key` в настройках YooKassa
6. Сохраните настройки: они будут храниться в базе данных

### Тестовый режим

Для тестирования используйте ключ с префиксом `test_` и сохраните его в той же секции панели.

### Webhook

ЮКасса отправляет уведомления об оплате на:
```
https://your-domain.com/api/v1/payments/webhook/yookassa
```

Настройте этот URL в личном кабинете ЮКассы.

### Проверка оплаты вручную

Пользователь может нажать кнопку «🔄 Проверить оплату» в боте.

## Telegram Stars ⭐

Встроенная платёжная система Telegram. Настраивается через панель и не требует отдельных секретов в `.env`.

### Конвертация

По умолчанию: **1 ₽ = 1 Star** (настраивается в коде).

### Ограничения

- Минимум 1 Star
- Работает только в личных чатах с ботом

## Баланс

Внутренний счёт пользователя. Пополняется:
- Администратором через панель
- Промокодом типа `balance`
- Реферальным бонусом

### Автопродление

Если у пользователя включено автопродление и достаточно средств на балансе — при истечении подписки система автоматически:
1. Списывает стоимость тарифа
2. Продлевает ключ в Marzban
3. Отправляет уведомление

---

## Промокоды

Типы промокодов:

| Тип | Описание |
|-----|----------|
| `discount` | Скидка в % на следующую покупку |
| `balance` | Пополнение баланса в рублях |
| `days` | Добавление дней к подписке |

Создание в панели: **Промокоды → Создать**

Или в боте: `/promo CODE TYPE VALUE [MAX_USES]`
```
/promo SALE20 discount 20 100
/promo GIFT100 balance 100
/promo WEEK7 days 7 50
```

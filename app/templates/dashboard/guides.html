📲 Как подключиться

Android

  1. Скачайте приложение v2rayNG или Hiddify из Google Play
  2. Откройте приложение и нажмите "+"
  3. Выберите "Import from Clipboard"
  4. Скопируйте ссылку из раздела «Подписки» и вставьте
  5. Подключитесь

Windows / macOS / Linux

  1. Скачайте Nekoray или v2rayN с официального сайта
  2. Скопируйте ссылку из раздела «Подписки»
  3. В приложении нажмите "Import from Clipboard"
  4. Активируйте профиль

iOS

  1. Скачайте Shadowrocket или V2Box из App Store
  2. Скопируйте ссылку из раздела «Подписки»
  3. Добавьте конфигурацию через импорт
  4. Подключитесь
